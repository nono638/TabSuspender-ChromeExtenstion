📝 STEP-BY-STEP INSTRUCTIONS
═══════════════════════════════════════════════════════════════

METHOD 1: UPLOAD VIA GITHUB WEBSITE (EASIEST)

1. Go to your blank GitHub repo

2. Click "uploading an existing file"

3. Download these 3 files from Claude:
   - LICENSE (no extension)
   - README.md
   - All tab-suspender files EXCEPT generate-icons.js

4. Edit the LICENSE file:
   - Change [Your Name] to your actual name
   - Save it

5. Edit the README.md file:
   - Replace YOUR_USERNAME with your GitHub username
   - Replace YOUR_EXTENSION_ID with extension ID (get after publishing)
   - Replace your.email@example.com with your email
   - Save it

6. Drag and drop ALL files into GitHub
   - Maintain folder structure (icons/, modules/)
   - Make sure LICENSE and README.md are in root

7. Add commit message: "Initial commit - Intelligent Tab Suspender v1.6.2"

8. Click "Commit changes"

═══════════════════════════════════════════════════════════════

METHOD 2: UPLOAD VIA GIT COMMAND LINE

1. Download all files to a folder on your computer

2. Open terminal/command prompt in that folder

3. Run these commands:
   git init
   git add .
   git commit -m "Initial commit - Intelligent Tab Suspender v1.6.2"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/intelligent-tab-suspender.git
   git push -u origin main

4. Enter your GitHub credentials when prompted

🎯 AFTER UPLOAD
═══════════════════════════════════════════════════════════════

Your repo will be live at:
https://github.com/YOUR_USERNAME/intelligent-tab-suspender

THEN:
1. Get your extension published on Chrome Web Store
2. Get the extension ID from the store URL
3. Go back to GitHub and edit README.md
4. Replace YOUR_EXTENSION_ID with actual ID
5. Commit the change

EXTENSION ID LOCATION:
After publishing, your store URL will be:
https://chrome.google.com/webstore/detail/EXTENSION_ID_HERE

Copy the EXTENSION_ID_HERE part and use it in README.md

═══════════════════════════════════════════════════════════════

🎉 UPDATE YOUR CHROME WEB STORE LISTING
═══════════════════════════════════════════════════════════════

After uploading to GitHub:

1. Go back to Chrome Web Store Developer Dashboard

2. Edit your extension listing

3. Add to description:
   "• Open source - View code on GitHub: 
    https://github.com/YOUR_USERNAME/intelligent-tab-suspender"

4. Save changes
