METHOD 1: UPLOAD VIA GITHUB WEBSITE (EASIEST)

1. Go to your blank GitHub repo

2. Click "uploading an existing file"

3. Download these 3 files from Claude:
   - LICENSE (no extension)
   - README.md
   - All tab-suspender files EXCEPT generate-icons.js

4. Edit the LICENSE file:
   - Change [Your Name] to your actual name
   - Save it

5. Edit the README.md file:
   - Replace YOUR_USERNAME with your GitHub username
   - Replace YOUR_EXTENSION_ID with extension ID (get after publishing)
   - Replace your.email@example.com with your email
   - Save it

6. Drag and drop ALL files into GitHub
   - Maintain folder structure (icons/, modules/)
   - Make sure LICENSE and README.md are in root

7. Add commit message: "Initial commit - Intelligent Tab Suspender v1.6.2"

8. Click "Commit changes"
